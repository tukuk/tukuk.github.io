Utility AI Sempurna

Engine percuma dalam browser

Versi ini tidak bergantung pada API berbayar. Ia menggunakan engine ringkas client-side: pengesanan kategori, template generator, random idea engine, prompt builder, planner, dan mini coder untuk hasilkan output terus dalam chat.

Music engine

Hasilkan konsep lagu, lirik, chord, struktur verse-chorus, dan prompt muzik.

Visual engine

Hasilkan prompt gambar, konsep logo, palet warna, gaya visual, dan idea lukisan.

Coding engine

Hasilkan struktur fail, contoh HTML/CSS/JS, checklist debugging, dan pelan projek.

Open-source tools

Gunakan Web Audio API, Canvas, Speech API, localStorage, dan download chat tanpa kos.

Student dashboard

Pelan belajar, kuiz, flashcards, mode fokus, dan eksport portfolio untuk pelajar.

Utility sempurna

Tema, simpan template, salin jawapan, screenshot, eksport semua, dan preset pelajar.

Pelankan projek

Gunakan pelan ini untuk kembangkan projek dari demo statik kepada AI studio sebenar.

Roadmap ringkas

  • 1. Tambah halaman baharu seperti portfolio, dokumentasi, dan galeri demo.
  • 2. Sambung API AI melalui Cloudflare Worker atau backend kecil supaya API key tidak terdedah.
  • 3. Simpan sejarah chat dalam localStorage untuk pengalaman lebih baik.
  • 4. Tambah tema terang/gelap, carian, dan halaman admin ringkas.
